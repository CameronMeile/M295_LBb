# Dependencies to Install

- npm i express
- npm i express-session
- npm i swagger-ui-express
- npm i swagger-autogen
- npm i crypto
- npm i nodemon (optional)
  
### All in One
```
npm i express express-session swagger-ui-express swagger-autogen crypto nodemon
```
